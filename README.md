# khadijaTN

Personal portfolio / landing page built with React 18, TypeScript, Vite, and Tailwind CSS. Features animated particle backgrounds and smooth page transitions.

🔗 **Live:** [f-tnana.com](https://f-tnana.com)

## Tech Stack

| Tech | Version | Role |
|------|---------|------|
| React | 18.3 | UI framework |
| TypeScript | 5.x | Type safety |
| Vite | 5.4 | Build tool & dev server |
| Tailwind CSS | 3.4 | Styling |
| Framer Motion | 11.x | Animations & transitions |
| tsParticles | 3.5 | Particle background effects |
| React Router | 6.28 | Client-side routing |
| Lucide React | 0.454 | Icon library |

## Getting Started

### Prerequisites

- Node.js 18+
- npm or pnpm

### Installation

```bash
git clone https://github.com/dnxtpro/khadijaTN.git
cd khadijaTN
npm install
```

### Development

```bash
npm run dev      # Start dev server at http://localhost:5173
npm run build    # Production build
npm run preview  # Preview production build locally
npm run lint     # Run ESLint
```

## Deployment

Deployed automatically to Vercel on every push to `main`.
